# utmz

## installation

```
go get -u github.com/iwpnd/utmz
```

## usage

```
utmz --help
```

```
Usage: utmz [--lat latitude] [--lng longitude]

Basic Options:
--lat latitude : Latitude of Point to get EPSG code for
--lng longitude : Longitude of Point to get EPSG code for
```

### Example

```
utmz --lat 52 --lng 13

>> EPSG:32633
```
